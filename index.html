Twitch.tv buddy list
This project was an intro to usage of a JSON API, in this case by taking a list of Twitch.tv usernames and displaying some information about them. Also allows you to filter by username and online status.

Tic-Tac-Toe
A totally unfair game of tic-tac-toe with an unbeatable AI. The graphics in this game are emoji so you may want to use an emoji-friendly browser. Game lets you play first if you lose, game plays first if you tie.

Wikipedia Search
A live-updating article title search using the Wikipedia JSON API. Will also display a random article on command. I call it the "Hello Kitty information getter" because Hello Kitty was all I could think about (and all I searched for) when testing it.

Weather
A little weather applet, again practicing JSON requests, uses your IP address to guess your location and allows unit-switching. Displays different graphics based on temperature.

Retro Calculator
This retro-style calculator is super cute and just as functional as any other calculator from the seventies. Allows chaining of operations just like a real calculator would. Is not solar powered, unfortunately (I don't think jQuery has caught up enough for that yet).

Pomodoro Timer
A cute timer for the pomodoro scheduling program. Input work time and break time and watch as the circle fills up with seconds of your life you'll never get back. Bonus: plays an annoying noise when it gets to zero!

Twitter API Abuser
A rough and dirty hacked-together way of working around Twitter API auth tokens to read a user's tweets. Pushes an RSS feed of my friend @adamjk's tweets through an RSS to JSON filter and displays a random one at the press of a button.

FreeCodeCampNews
Last and pretty much least, another JSON data parsing exercise. This one takes information from the FreeCodeCamp news page and, like, displays it. I guess.
